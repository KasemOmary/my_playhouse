-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-

I've been into Emulation for years - so what better choice for my foray into making
my own project than Intel's first microprocessor?  Great idea, huh!  Anyways - this
project represents what I do for fun on a slow weekend.  At latest release, the CPU
is approximately 33% emulated - with a number of opcodes implemented - and 
enough functionality to have some awesome ASM calculations done!  Woo!  

Have fun exploring the dawn of microprocessors with this little program, and play with 
some Intel assembler like it's 1971!

To learn more about the 4004 - visit http://en.wikipedia.org/wiki/Intel_4004

Enjoy!
